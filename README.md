# Gestion de Patrimoine Immobilier

Application de gestion de biens immobiliers (location nue, meublée, tourisme).

## Stack
- **Frontend** : Expo (React Native Web + Android)
- **Backend** : Supabase (PostgreSQL + Auth + API)
- **Déploiement web** : Vercel
- **Déploiement Android** : Google Play via EAS Build

## Modules prévus
- Biens immobiliers
- Locataires & contrats
- Suivi des loyers & paiements
- Charges & dépenses
- Fiscalité (micro-foncier, réel, LMNP, LMP)
- Documents (baux, états des lieux)
